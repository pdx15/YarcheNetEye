# Yarche Net Eye

Yarche Net Eye - программа для мониторинга сетевой активности на компьютере. Приложение показывает активные TCP/UDP-соединения по процессам: PID, имя процесса, тип соединения, локальные и удаленные адреса, порты, состояние и путь к исполняемому файлу.

Проект находится на ранней стадии разработки.

## Возможности

- Просмотр текущих TCP и UDP-соединений.
- Отображение информации о процессе для каждого соединения.
- Журнал новых подключений, появившихся после запуска программы.
- Поиск по данным таблицы.
- Фильтры:
  - только внешние публичные адреса;
  - тип соединения: TCP или UDP;
  - состояние соединения;
  - только соединения с удаленным адресом;
  - только `ESTABLISHED`.
- Автообновление и ручное обновление списка соединений.
